## Future — Progress Notes / EMR Module

| Component | Description |
|-----------|-------------|
| **HPI** | ICD-10 based, editable (similar to plan section of Emed). Includes redflag/worsening trends |
| **PMH** | ICD-10 list of medical history |
| **PSH** | ICD-10 list of surgical history |
| **Meds** | Medication list with contraindications/Beers list/picks up contraindications |
| **Allergies** | Allergy list |
| **ROS** | Review of Systems with symptom trends |
| **PE** | Physical Exam with vital trends |
| **Assessment** | ICD-10 list |
| **Plan** | Similar to HPI but modified for plans |
| **Geri** | Geriatric-specific |
| **Preventative** | Preventative care |
| **MIPS** | Merit-based Incentive Payment System |
| **AWV** | Annual Wellness Visit |
| **RPM** | Remote Patient Monitoring |
| **Cog** | Cognitive |
| **BH** | Behavioral Health |
