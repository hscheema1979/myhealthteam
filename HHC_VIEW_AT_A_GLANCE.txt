1. Export Today's Data
   → Click "📥 Download as CSV"
   → File: hhc_patients_YYYYMMDD.csv
   → Ready for Google Sheets

2. Find Unassigned Patients
   → Click "Assigned" column header
   → All "No" values group together

3. Review High-Risk Patients
   → Scroll right to "Risk" column
   → Click header to sort
   → High-risk at top

4. See Provider Distribution
   → Click "Prov" column header
   → Groups by provider

5. Update View with Latest Data
   → Click "🔄 Refresh Data"
   → Reloads from database
