DATABASE QUERY DETAILS
================================================================================

Tables Used:
  - patients (primary)
  - provider_tasks (provider assignments)
  - users (names for providers/coordinators)

Joins:
  - LEFT JOIN provider_tasks
  - LEFT JOIN users (for provider names)
  - LEFT JOIN users (for coordinator names)

Filters:
  - WHERE LOWER(p.status) = 'active'

Ordering:
  - ORDER BY p.last_name, p.first_name

NULL Handling:
  - COALESCE(pr.first_name || ' ' || pr.last_name, 'Unassigned')
