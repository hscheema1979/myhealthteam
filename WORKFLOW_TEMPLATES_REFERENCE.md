# Role Legend

| Role Code | Full Name | Description |
|-----------|-----------|-------------|
| **CC** | Care Coordinator | Primary coordinator for patient care coordination |
| **CM** | Case Manager / Coordinator Manager | Management-level coordination (Role 40) |
| **CP** | Care Provider | Medical provider who reviews results and delivers care |
| **LC** | Lead Coordinator | Lead coordinator role (Role 37) |
| **RR** | Results Reviewer | **NEW ROLE** - Reviews lab/imaging results and coordinates with CP |
| **POT** | Patient Onboarding Team | Handles new patient onboarding process |
